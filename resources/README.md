# CertMgr Script Lib

The script lib is a first version of a simple wrapper to use the export and import functionality which CertMgr provides.
On top of the basic import/export/create exportable key functionality provided by the cerstore.nsf GUI, some additional functions are added to store trusted roots in Domino Directory.

The code can be copied into a script lib and is implemented as a Lotus Script class.


See the source code of the script lib for details.
